Translating Natural Language to Linear Temporal Logic (LTL)

This provides instructions on setting up a local environment to translate natural language system specifications into LTL.

Prereqs
* Python 3.10 or higher installed locally.
* A valid OpenAI API Key.

Run the following commands in your local terminal to set up the workspace:

```bash
# clone the repo
git clone [https://github.com/your-username/llm-logic-research.git](https://github.com/your-username/llm-logic-research.git)
cd llm-logic-research

#  activate a virtual env
python3 -m venv venv
source venv/bin/activate

# install  dependencies
pip install -r requirements.txt
